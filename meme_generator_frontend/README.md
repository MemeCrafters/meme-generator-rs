# 表情包生成器 - 前端

基于 **Vue 3** + **Vite** + **Tailwind CSS** 的现代化表情包制作前端界面。

## 技术栈

- **Vue 3** (Composition API + `<script setup>`)
- **Vue Router 4** (路由管理)
- **TypeScript** (类型安全)
- **Vite 6** (构建工具)
- **Tailwind CSS 3** (原子化 CSS)

## 开发

### 安装依赖

```bash
cd frontend
npm install
```

### 启动开发服务器

```bash
npm run dev
```

开发服务器运行在 `http://localhost:3000`，自动代理 `/api` 请求到后端服务 `http://127.0.0.1:2233`。

> 请确保后端服务已启动（默认端口 2233）。

### 生产构建

```bash
npm run build
```

构建产物在 `dist/` 目录，可直接部署为静态文件。

## 功能

- **表情包浏览** — 网格布局展示所有可用模板，支持懒加载预览
- **搜索和筛选** — 按关键词、标签搜索表情包模板
- **表情包制作** — 上传图片、输入文字、调整选项，一键生成
- **结果下载** — 生成后可直接下载表情包图片

## 页面结构

- `/` — 表情包列表页（搜索、标签筛选、预览）
- `/meme/:key` — 表情包制作页（上传图片、输入文字、生成）

## API 代理配置

开发模式下，Vite 会将 `/api/*` 代理到后端服务：

| 前端路径 | 后端路径 |
|---------|---------|
| `/api/meme/keys` | `http://127.0.0.1:2233/meme/keys` |
| `/api/memes/{key}` | `http://127.0.0.1:2233/memes/{key}` |
| `/api/image/{id}` | `http://127.0.0.1:2233/image/{id}` |

生产部署时需要自行配置反向代理（如 Nginx）。
